# 🛒 Sistema de Vendas em C

## 📌 Sobre o projeto

Este projeto consiste em um sistema de vendas desenvolvido em linguagem C, com foco no gerenciamento de produtos e controle de vendas.

A aplicação permite realizar operações básicas de um sistema comercial, incluindo cadastro, exclusão de produtos e venda de itens.

---

## 🛠️ Tecnologias utilizadas

* Linguagem C
* Git
* GitHub

---

## ⚙️ Funcionalidades

* Cadastro de produtos
* Exclusão de produtos
* Realização de vendas

---

## ▶️ Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/lucasproencab98/sistema_vendas_C
```

### 2. Acessar a pasta do projeto

```bash
cd sistema_vendas_C
```

### 3. Compilar o programa

```bash
gcc vendas.c -o sistema
```

### 4. Executar

```bash
./sistema
```
