# 🛒 Sistema de Vendas em C

## 📌 Sobre o projeto

Este projeto consiste em um sistema de vendas desenvolvido em linguagem C, com foco no gerenciamento de produtos e controle de vendas.

A aplicação permite realizar operações básicas de um sistema comercial, incluindo cadastro, exclusão de produtos e geração de recibos. 
Além disso, os dados são persistidos em arquivo, garantindo que as informações não sejam perdidas ao fechar o sistema.

## 🛠️ Tecnologias utilizadas

* Linguagem C
* Git
* GitHub

---

## ⚙️ Funcionalidades

* Cadastro de produtos
* Exclusão de produtos
* Realização de vendas
* Geração de recibo de vendas
* Persistência de dados em arquivo (salvamento automático)

---

## ▶️ Como executar o projeto

# Clonar o repositório
git clone https://github.com/lucasproencab98/learningc

# Acessar a pasta do projeto
cd learningc

# Compilar o programa
gcc main.c -o sistema 

# Executar 
./sistema
